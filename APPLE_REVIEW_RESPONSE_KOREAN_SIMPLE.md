## 📋 제출 방법

### 1단계: App Store Connect 접속
1. https://appstoreconnect.apple.com 로그인
2. "나의 앱" → "Tarot Timer" 클릭
3. 좌측 메뉴에서 "버전 1.1.3" 클릭
4. **"Resolution Center"** 또는 **"심사 정보"** 섹션 찾기

### 2단계: 회신 작성
1. "Reply to App Review" 또는 "심사팀에 답변" 버튼 클릭
2. 위의 회신 내용 복사
3. 붙여넣기
4. "Send" 또는 "보내기" 클릭
