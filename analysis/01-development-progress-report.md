- **영향**: Build 114, 115, 116, 117 모두 동일 증상
- **사용자 피드백**: 실제 사용자가 구독 불가능 상태

### 2. **시도한 해결 방법들**
- ❌ **Build 117**: StoreKit2 모드 명시적 활성화 → 실패
- 🔄 **현재 접근**: 라이브러리 기본 설정 사용 + 상세 디버그 로깅 (코드 준비 완료, 빌드 대기)

### 3. **Android 빌드 업데이트 완료** ✅
- Android versionCode: 103 → 104
- 로컬 AAB 빌드 성공 (122MB)
- 최근 iOS 수정사항 모두 포함

- 24-48시간 전파 지연이 가장 흔한 원인
