#### 🔥 **즉시 처리 필요** (알림 시스템 완전 분리 완료된 상태에서 App Store 제출)
1. **iPad 스크린샷 촬영 및 업로드**
   - **목표**: App Store Connect 제출 요구사항 완료
   - **기능**:
     - 13개 필수 iPad 스크린샷 촬영 (2048x2732 해상도)
     - 주요 기능별 화면 캡처 (알림 설정, 조용한 시간, 카드 뽑기, 스프레드 등)
     - 다국어 UI 스크린샷 (한/영)
     - App Store Connect 메타데이터 입력 완료
   - **예상 소요**: 1일
   - **담당 영역**: App Store 제출 0% → 100%

2. **App Store Connect 실제 제출**
   - **목표**: Build 16 기준 심사 신청
   - **기능**:
     - NSUserTrackingUsageDescription 추적 권한 설정 ("데이터 추적: 아니요")
     - 심사 위원 메시지 입력 (한글/영문)
     - 개인정보 처리방침 최종 검토
     - 제출 및 심사 대기
   - **예상 소요**: 1일
   - **담당 영역**: App Store 제출 0% → 100%

3. **심사 대응 준비**
   - **목표**: 심사 피드백에 신속 대응
   - **기능**:
     - 심사 위원 질문 대응 시나리오 준비
     - 교육용 앱 포지셔닝 근거 자료 정리
     - 추가 요구사항 대비 빌드 준비
   - **예상 소요**: 1일
   - **담당 영역**: 심사 대응 0% → 100%
