Tarot Timer - 24시간 타로 카드 학습 앱

타로카드 78장을 시간 기반 학습 시스템으로 마스터하세요. 매 시간 새로운 카드를 만나며 자연스럽게 배웁니다.

24시간 학습 시스템

시간 연동 학습
매 정각마다 새로운 카드 학습 세션이 시작됩니다. 하루 24장 반복 학습으로 자연스럽게 암기하며, 간격 반복 학습 원리가 자동으로 적용됩니다.

자동 스케줄러
자정 00시에 자동 초기화되고, 다음 학습 시간을 자동으로 알려줍니다. 백그라운드에서도 시간을 정확하게 추적합니다.

학습 다이어리
시간별 카드 기록이 자동 저장되며, 개인 메모를 작성할 수 있습니다. 과거 학습 내역을 언제든지 확인할 수 있습니다.

완전한 타로 카드 데이터

메이저 아르카나 22장과 마이너 아르카나 56장(완드, 컵, 소드, 펜타클)의 정방향과 역방향 의미를 모두 제공합니다. 카드별 상징과 키워드, 서양 상징학 기반 해석이 포함되어 있습니다.

다국어 지원

한국어, 영어, 일본어 3개 언어를 지원합니다. 한국어는 전통적 타로 해석, 영어는 국제 표준 용어, 일본어는 동양적 관점의 해석을 제공합니다.

주요 기능

시간 기반 자동 학습 시스템, 78장 완전 데이터베이스, 학습 일지 및 메모, 스마트 알림, 3개 언어 지원, 다크 모드 UI, 오프라인 사용 가능, 자동 저장, 개인정보 보호(로컬 저장)

교육 목적

이 앱은 점술이나 운세 제공이 아닌, 타로 카드의 상징과 의미를 배우는 교육 도구입니다. 서양 상징학 학습, 문화적 이해 도구, 자기성찰 및 개인 성장을 위한 앱입니다.
