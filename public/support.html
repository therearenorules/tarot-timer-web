❓ 자주 묻는 질문 / FAQ

Q1. 구독을 취소하려면 어떻게 해야 하나요?
iOS: 설정 → Apple ID → 구독 → 타로 타이머 → 구독 취소
Android: Google Play Store → 계정 → 결제 및 정기 결제 → 타로 타이머 → 취소
Q2. 환불은 어떻게 받나요?
iOS: reportaproblem.apple.com에서 신청
Android: Google Play 지원 센터에서 환불 요청
구독 후 7일 이내에는 플랫폼 정책에 따라 환불이 가능합니다.
Q3. 구매 내역을 다른 기기에서도 사용할 수 있나요?
네! 같은 Apple ID 또는 Google 계정을 사용하는 모든 기기에서 프리미엄 기능을 이용하실 수 있습니다. 앱 설정에서 "구매 복원" 버튼을 눌러주세요.
Q4. 알림이 오지 않아요.
다음 사항을 확인해주세요:
1. 기기 설정 → 알림 → 타로 타이머 → 알림 허용 확인
2. 앱 설정 → 알림 설정 → 시간별 알림 활성화 확인
3. "조용한 시간" 설정이 켜져있지 않은지 확인
Q5. 데이터를 백업하고 싶어요.
프리미엄 사용자는 설정 → 클라우드 백업에서 자동 백업을 활성화할 수 있습니다. 무료 사용자는 기기 내부에만 데이터가 저장되며, 앱 삭제 시 데이터도 함께 삭제됩니다.
Q6. 언어를 변경하려면?
앱 설정 → 언어 설정에서 한국어, 영어, 일본어 중 선택할 수 있습니다.

❓ Frequently Asked Questions (English)

Q1. How do I cancel my subscription?
iOS: Settings → Apple ID → Subscriptions → Tarot Timer → Cancel
Android: Google Play Store → Account → Payments & subscriptions → Cancel
Q2. How do I request a refund?
iOS: Visit reportaproblem.apple.com
Android: Request refund through Google Play Support
Refunds are available within 7 days according to platform policies.
Q3. Can I use my purchase on multiple devices?
Yes! You can use premium features on all devices with the same Apple ID or Google account. Just tap "Restore Purchases" in app settings.
Q4. I'm not receiving notifications.
Please check:
1. Device Settings → Notifications → Tarot Timer → Allow Notifications
2. App Settings → Notifications → Enable Hourly Notifications
3. Make sure "Quiet Hours" is not enabled
Q5. How do I backup my data?
Premium users can enable automatic cloud backup in Settings → Cloud Backup. Free users' data is stored locally and will be deleted if the app is uninstalled.
