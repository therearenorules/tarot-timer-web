# 안드로이드 앱 출시 기획서 작성 요청

## 목표
현재 iOS 앱스토어에 출시된 앱을 분석하고, 안드로이드 플랫폼 출시를 위한 포괄적인 기획 전략을 수립해주세요.

## 분석 및 작업 요청사항

### 1. 현재 앱 분석
- 프로젝트의 모든 파일을 분석하여 앱의 주요 기능, 화면 구성, 기술 스택을 파악
- iOS 앱의 화면 크기 및 레이아웃 구조 분석
- 사용 중인 라이브러리 및 의존성 확인
- 현재 수익 모델 및 인앱 결제 구조 파악

### 2. 안드로이드 출시 기획서 작성
다음 내용을 포함한 상세한 마크다운 문서를 `ANDROID_LAUNCH_PLAN.md` 파일로 생성해주세요:

#### A. 기술적 요구사항
- **반응형 디자인 전략**
  - 다양한 안드로이드 화면 크기 대응 방안 (폰, 태블릿, 폴더블)
  - 화면 밀도(density) 별 대응 전략 (mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)
  - 구체적인 브레이크포인트 설정 및 레이아웃 조정 계획
  - Safe Area 및 노치/펀치홀 대응 방안

- **플랫폼별 차이점 대응**
  - iOS와 안드로이드 UI/UX 가이드라인 차이점
  - 네비게이션 패턴 조정 필요사항
  - 권한 처리 방식 차이
  - 백그라운드 동작 및 알림 처리 차이점

#### B. 광고 통합 전략
- **광고 플랫폼 선택 및 설정**
  - 추천 광고 플랫폼 (Google AdMob, Facebook Audience Network 등)
  - 각 플랫폼의 장단점 비교
  - 광고 유형 추천 (배너, 전면, 리워드, 네이티브)
  - 광고 배치 위치 및 UX 최적화 방안
  
- **광고 계정 설정 가이드**
  - Google AdMob 계정 생성 단계별 가이드
  - 앱 등록 및 광고 단위 생성 방법
  - 테스트 광고 vs 실제 광고 설정
  - 광고 SDK 통합 코드 예시

- **광고 수익 최적화**
  - 광고 표시 빈도 및 타이밍 권장사항
  - 사용자 경험을 해치지 않는 광고 전략
  - A/B 테스트 계획

#### C. 사업자 등록 및 법적 요구사항
- **Google Play Console 설정**
  - 개발자 계정 등록 절차 (일회성 $25 등록비)
  - 필요 서류 및 정보 목록
  - 결제 프로필 설정 방법

- **사업자 등록 요구사항**
  - 대한민국 사업자등록 필요 여부 및 절차
  - 개인 개발자 vs 사업자 등록의 차이점
  - 세금 관련 정보 (VAT, 소득세 등)

- **개인정보 처리방침 및 약관**
  - 안드로이드 앱에 필요한 개인정보 처리방침 작성 가이드
  - 광고 관련 개인정보 수집 고지사항
  - GDPR 및 개인정보보호법 준수 사항

#### D. 버전 전략
- **무료 vs 프리미엄 구분**
  - 무료 버전 기능 범위
  - 광고 제거 옵션 (인앱 구매)
  - 프리미엄 기능 제안
  - 가격 책정 전략

#### E. 출시 준비 체크리스트
- **개발 단계**
  - [ ] 반응형 레이아웃 구현 및 다양한 디바이스 테스트
  - [ ] 광고 SDK 통합 및 테스트
  - [ ] 인앱 결제 (광고 제거) 구현
  - [ ] 안드로이드 고유 기능 대응
  - [ ] 성능 최적화 및 메모리 관리

- **스토어 준비**
  - [ ] 앱 아이콘 (다양한 해상도)
  - [ ] 스크린샷 (폰/태블릿, 최소 2개 이상)
  - [ ] 기능 그래픽 (1024x500)
  - [ ] 앱 설명 (짧은 설명, 전체 설명)
  - [ ] 개인정보처리방침 URL
  - [ ] 콘텐츠 등급 설정

- **계정 및 설정**
  - [ ] Google Play Console 개발자 계정
  - [ ] AdMob 계정 및 앱 등록
  - [ ] 앱 서명 키 생성 및 관리
  - [ ] 결제 프로필 설정
  - [ ] 사업자 정보 등록 (필요시)

#### F. 테스트 계획
- **디바이스 테스트 매트릭스**
  - 테스트해야 할 주요 디바이스 목록
  - 화면 크기별 테스트 시나리오
  - OS 버전별 호환성 테스트
  - 광고 표시 및 인터랙션 테스트

- **베타 테스트 전략**
  - 내부 테스트 → 비공개 테스트 → 공개 테스트 단계
  - 테스터 모집 및 피드백 수집 방법

#### G. 출시 일정
- 각 단계별 예상 소요 시간
- 주요 마일스톤 설정
- 출시 전후 모니터링 계획

#### H. 예상 비용
- 개발자 계정 등록비
- 광고 플랫폼 수수료 구조
- 스토어 수수료 (매출의 15-30%)
- 기타 예상 비용

### 3. 추가 요청사항
- 모든 정보는 2025년 최신 기준으로 작성
- 실제 실행 가능한 구체적인 가이드 제공
- 각 단계별 참고 링크 및 문서 첨부
- 한국 개발자 기준으로 작성 (한국어 문서, KRW 기준 등)
- 잠재적인 문제점과 해결 방안 포함

필요시 다음 항목도 생성해주세요:
- 광고 통합을 위한 샘플 코드
- 반응형 레이아웃 구현 예시
- 화면 크기별 설정 파일 (dimensions.xml, layouts 등)
- 광고 로딩 및 에러 핸들링 코드
